== Description ==

A WordPress plugin that allows you to use Twitters Bootstrap buttons in your content.

The plugin offers 3 buttons sizes (Large, Small and Mini), 7 Button styles/colors (Default, Primary, Info, Success, Warning, Danger and Inverse) and 140 Icons in both Black and White versions.

This gives you a total 5091 variations to use. Thatâ€™s pretty impressive!

Includes a shortcode inserter to make inserting your buttons hassle free in both the visual and html editor.

[Documentation](http://cramer.co.za/bootstrap-buttons/)


== Installation ==

1. Upload the plugin folder 'bootstrap-buttons' to your `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A Bootstrap Buttons button will be available above the Content Editor when writing a post or page.
